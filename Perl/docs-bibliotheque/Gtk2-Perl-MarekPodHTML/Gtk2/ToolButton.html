PROPERTIES

'icon-name' (string : readable / writable / private)
The name of the themed icon displayed on the item
'icon-widget' (Gtk2::Widget : readable / writable / private)
Icon widget to display in the item
'label' (string : readable / writable / private)
Text to show in the item.
'label-widget' (Gtk2::Widget : readable / writable / private)
Widget to use as the item label
'stock-id' (string : readable / writable / private)
The stock icon displayed on the item
'use-underline' (boolean : readable / writable / private)
If set, an underline in the label property indicates that the next character should be used for the mnemonic accelerator key in the overflow menu
