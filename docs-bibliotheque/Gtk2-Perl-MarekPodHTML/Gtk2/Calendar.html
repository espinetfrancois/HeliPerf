PROPERTIES

'day' (integer : readable / writable / private)
The selected day (as a number between 1 and 31, or 0 to unselect the currently selected day)
'month' (integer : readable / writable / private)
The selected month (as a number between 0 and 11)
'no-month-change' (boolean : readable / writable / private)
If TRUE, the selected month cannot be changed
'show-day-names' (boolean : readable / writable / private)
If TRUE, day names are displayed
'show-heading' (boolean : readable / writable / private)
If TRUE, a heading is displayed
'show-week-numbers' (boolean : readable / writable / private)
If TRUE, week numbers are displayed
'year' (integer : readable / writable / private)
The selected year
