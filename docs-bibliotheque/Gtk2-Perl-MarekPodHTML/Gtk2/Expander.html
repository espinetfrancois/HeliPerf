PROPERTIES

'expanded' (boolean : readable / writable / construct / private)
Whether the expander has been opened to reveal the child widget
'label' (string : readable / writable / construct / private)
Text of the expander's label
'label-widget' (Gtk2::Widget : readable / writable / private)
A widget to display in place of the usual expander label
'spacing' (integer : readable / writable / private)
Space to put between the label and the child
'use-markup' (boolean : readable / writable / construct / private)
The text of the label includes XML markup. See pango_parse_markup()
'use-underline' (boolean : readable / writable / construct / private)
If set, an underline in the text indicates the next character should be used for the mnemonic accelerator key
