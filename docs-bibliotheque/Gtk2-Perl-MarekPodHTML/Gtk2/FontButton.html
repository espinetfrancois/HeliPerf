PROPERTIES

'font-name' (string : readable / writable / private)
The name of the selected font
'show-size' (boolean : readable / writable / private)
Whether selected font size is shown in the label
'show-style' (boolean : readable / writable / private)
Whether the selected font style is shown in the label
'title' (string : readable / writable / private)
The title of the font selection dialog
'use-font' (boolean : readable / writable / private)
Whether the label is drawn in the selected font
'use-size' (boolean : readable / writable / private)
Whether the label is drawn with the selected font size
