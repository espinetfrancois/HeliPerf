NAME

Gtk2::ItemFactory

HIERARCHY

  Glib::Object
  +----Glib::InitiallyUnowned
       +----Gtk2::Object
            +----Gtk2::ItemFactory

DEPRECATION WARNING

Gtk2::ItemFactory has been marked as deprecated, and should not be used in newly written code.

You should use Gtk2::UIManager instead of Gtk2::ItemFactory.
